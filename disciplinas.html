Exemplo de Algoritmo de Ordenação

O algoritmo de ordenação por bolha (Bubble Sort) é um dos algoritmos de ordenação mais simples:


Algoritmo BubbleSort
    Entrada: Lista de números
    Saída: Lista ordenada

    n ← comprimento da lista
    Para i ← 0 até n - 2 faça
        Para j ← 0 até n - i - 2 faça
            Se lista[j] > lista[j + 1] então
                Trocar lista[j] e lista[j + 1]
            FimSe
        FimPara
    FimPara
    Retornar lista

Exemplo de Algoritmo de Busca

O algoritmo de busca binária é utilizado para encontrar um elemento em uma lista ordenada:


Algoritmo BuscaBinaria
    Entrada: Lista ordenada e valor a ser encontrado
    Saída: Posição do valor ou -1 se não encontrado

    Início ← 0
    Fim ← comprimento da lista - 1

    Enquanto Início ≤ Fim faça
        Meio ← (Início + Fim) / 2
        Se lista[Meio] = valor então
            Retornar Meio
        SenãoSe lista[Meio] < valor então
            Início ← Meio + 1
        Senão
            Fim ← Meio - 1
        FimSe
    FimEnquanto

    Retornar -1

Programação em C

Aqui estão alguns conceitos e exemplos básicos de programação em C.

Exemplo de Código C: Calculadora Simples

Este código C mostra uma calculadora simples que realiza operações básicas:


#include <stdio.h>

int main() {
    int a, b;
    char op;
    
    printf("Digite dois números e uma operação (+, -, *, /): ");
    scanf("%d %d %c", &a, &b, &op);
    
    switch(op) {
        case '+':
            printf("Resultado: %d\n", a + b);
            break;
        case '-':
            printf("Resultado: %d\n", a - b);
            break;
        case '*':
            printf("Resultado: %d\n", a * b);
            break;
        case '/':
            printf("Resultado: %d\n", a / b);
            break;
        default:
            printf("Operação inválida\n");
    }
    
    return 0;
}

Exemplo de Código C: Vetores

Este código C demonstra como manipular vetores:


#include <stdio.h>

int main() {
    int vetor[5] = {1, 2, 3, 4, 5};
    int i;

    printf("Conteúdo do vetor:\n");
    para (i = 0; i < 5; i++) {
        printf("%d ", vetor[i]);
    }
    printf("\n");

    return 0;
}

Banco de Dados

Nesta seção, discutimos conceitos básicos de banco de dados e SQL.

Exemplo de Consulta SQL

Este exemplo mostra uma consulta SQL básica para selecionar dados de uma tabela:


SELECT * FROM usuarios WHERE idade > 18;

Essa consulta retorna todos os usuários com mais de 18 anos.

Exemplo de Criação de Tabela SQL

Este exemplo mostra como criar uma tabela em SQL:


CREATE TABLE usuarios (
    id INT AUTO_INCREMENT PRIMARY KEY,
    nome VARCHAR(100),
    idade INT
);
